# Tracking Food Calories App

A simple app to track foods and their calories so users can monitor daily intake and reach nutrition goals.

## Features
- Add, edit, and remove food entries
- Track calories per meal and per day
- View daily/weekly summaries
- Search and categorize foods
- Export data (CSV)

## Potential Tech Stack
- Frontend: React / React Native
- Backend: Node.js + Express or Python (FastAPI)
- Database: PostgreSQL or SQLite (for small/local use)
- Auth: OAuth / JWT
- Storage: Optional (for images)

## Getting started (local development)
1. Clone the repo:
   ```bash
   git clone https://github.com/Barrientosgers/tracking-food-calories-app.git
   ```
2. Install dependencies (example for Node.js):
   ```bash
   cd tracking-food-calories-app
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
